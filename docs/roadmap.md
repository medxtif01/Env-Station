# Roadmap — Env-Station

## Phase 1 — Mise en place du projet (terminé)
- Création du dépôt.
- Organisation des répertoires.
- Rédaction de la documentation initiale.

## Phase 2 — Firmware
- Lecture du BME680.
- Calibration de base.
- Organisation des données en JSON.
- Ajout de la gestion des erreurs I2C.

## Phase 3 — Intégration cloud
- Choix de la plateforme (Firebase, MQTT, API REST).
- Envoi des données via Wi-Fi.
- Développement d’un tableau de bord minimal.

## Phase 4 — Documentation
- Ajout des schémas matériels.
- Ajout des diagrammes.
- Rédaction d’un document de synthèse.
