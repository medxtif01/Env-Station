# Env-Station
IoT Environmental Monitoring Station (ESP32, BME680)
Env-Station – Environmental Monitoring System

Système de mesure environnementale basé sur ESP32 + BME680

Description du projet

Env-Station est une station environnementale IoT capable de mesurer :

  Température
  Humidité
  Pression atmosphérique
  Qualité de l’air (COV, IAQ)

Les données sont collectées via un ESP32 et transmises à une interface cloud (future version).

 Objectifs du projet

 Concevoir un système embarqué complet
 Tester et calibrer le capteur BME680
 Développer un firmware simple et robuste
 Préparer une extension cloud (MQTT / API Rest / Firebase)
