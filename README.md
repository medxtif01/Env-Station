# Env-Station — Environmental Monitoring System

## Description du projet
Env-Station est une station de mesure environnementale basée sur un microcontrôleur ESP32 et un capteur BME680. Le système permet la mesure de la température, de l’humidité, de la pression atmosphérique et de la qualité de l’air (composés organiques volatils).

## Objectifs
- Développer un système embarqué complet et fiable.
- Implémenter un firmware assurant l'acquisition et le traitement des données.
- Préparer une future extension vers une interface cloud pour la visualisation en ligne.
- Documenter l’architecture matérielle et logicielle.
